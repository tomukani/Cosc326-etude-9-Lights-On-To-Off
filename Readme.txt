To compile our program navigate to folder location in terminal and enter command
javac *.java

To run our program navigate to folder location in terminal and enter command
java Main
This will prompt you to enter a set of switches and then enter a set of connections and will output a solution to turn all the lights off.

To input a text file containing different switch and connection combinations enter command
java Main < <filename>.txt
